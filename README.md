## GET vs POST — What is the Difference?

### GET Method
Sends data **in the URL**
 Data is **visible** to everyone
 Example: `https://website.com?name=Feranmi`
 Used for **search forms**
 Not secure for personal data

### POST Method
- Sends data **hidden in the request body**
- Data is **NOT visible** in the URL
- URL stays as `https://website.com`
- Used for **contact forms, login, signup**
- More secure for personal data

### Which I Used
I used **POST** in this project because
the form collects personal information
like name and email which should not
be visible in the URL.
